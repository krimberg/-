# Классификация состояния растений по мультиспектральным снимкам с дронов

**Бинарная и многоклассовая классификация** (здоровое / больное / стресс от засухи / удобрений и т.д.)

### Используемые технологии
- Модель: **ResNet50** (можно заменить на EfficientNet или Vision Transformer)
- Фреймворк: PyTorch
- Данные: PlantVillage (RGB) или multispectral drone imagery

**Выполненные требования:**
- Предобработка изображений (аугментация + нормализация)
- Обучение с валидацией (train/val split)
- Полный отчёт: Accuracy, F1-score, Confusion Matrix + графики
